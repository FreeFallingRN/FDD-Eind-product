Front-end Uploaden Interaction Design

Uploaden

Mijn website staat online, waarschijnlijk is dit hoe je hem op het moment aan het bekijken bent. Het adres van mijn homepagina is: https://fdd.che-ict.nl/2122/bjvos/

Om een website online te brengen moet je zorgen dat al je bestanden op de server van het domein komen te staan, zelf gebruik ik hiervoor Cloudberry, dit is te downloaden via de volgende link: https://www.msp360.com/explorer.aspx

Als je server een Amazon S3 server is kan je de volgende stappen doorlopen om je website online te brengen.

  1. Maak een nieuw Amazon S3 storage account aan
  2. Voer de weergavenaam naar keuze in bij Display name
  3. Voer de Acces en Secret key in, deze heb je gekregen van de beheerder om bestanden te kunnen uploaden
  4. Test of alles werkt met ‘Test connection’, bij succes krijg je de groene vink te zien
  5. Links selecteer je waar de bestanden van jouw website staan
  6. En rechts selecteer je het profiel dat je net hebt aangemaakt en ga je naar de URL van jouw website
  7. Sleep de bestanden van links naar rechts en je website staat online
