Framework

Deze hele website is door mij persoonlijk ontworpen en gescript in HTML en CSS. Het eigendom van externe bronnen is de 404 pagina die je misschien al bent tegengekomen. Ik had een framework kunnen gebruiken, maar op het moment dat ik dat hoorde had ik al mijn volledig ontwerp, de structuur en opmaak van mijn website af. Als ik wel een framework had gebruikt had dat me waarschijnlijk een aantal uur gescheeld, ik heb er nu echter wel een beetje ervaring voor teruggekregen.

Wireframe

Als ik het wireframe naast de website houd zie ik dat de header het enige is dat compleet hetzelfde is gebleven. Dat is voor mij leuk om te zien omdat dat ook het eerste deel is dat ik heb ontworpen in mijn paper prototype. Dan heb ik op de homepagina mijn kaartjes, op de website zijn deze hetzelfde gebleven, ik ben alleen overgestapt op een ander kleurenschema omdat de oude kleuren niet bij mijn achtergrond pasten. Ook het stukje introductie op de homepagina is niet verandert.

Van de detailpagina is vrij weinig overgebleven. De inhoud is nog steeds precies hetzelfde, maar alles is minder breed. Dit maakt het lezen wat fijner.

Mijn contactpagina is bijna hetzelfde gebleven, alleen de kleur is aangepast om meer bij de achtergrond te passen en de hoekjes zijn rond. De header met het stukje tekst daaronder staan er nog steeds.

Commentaar

Ik heb in mijn code regelmatig wat commentaar toegevoegd, vooral om aan te geven wat in de HTML wat op de website is. In het CSS bestand heb ik juist commentaar kopjes gebruikt om te ordenen welke CSS op welk stuk HTML invloed had.

De source code van deze website is online te vinden in mijn Github rep.
